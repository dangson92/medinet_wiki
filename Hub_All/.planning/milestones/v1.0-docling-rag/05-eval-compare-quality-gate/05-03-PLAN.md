**Verdict logic:**
- `native_rank` ≤ top-3 AND `docling_rank` ≤ top-3 → `unchanged`
- `native_rank` is null (or > 3) AND `docling_rank` ≤ top-3 → `FIXED`
- `native_rank` ≤ top-3 AND (`docling_rank` is null OR > 3) → `REGRESSED`
- Cả 2 null → `both_miss`
- Còn lại (cả 2 > 3 nhưng có rank) → `both_miss_top3`
