Tạo `eval/run_extraction_compare.py`:

1. **Argparse:** `--native eval/baseline_native.json`, `--docling eval/baseline_docling.json`, `--output eval/extraction_compare.json`, `--db-dsn` (default từ env).

2. **Pre-condition check:** 2 file snapshot tồn tại + đọc được. Nếu thiếu `baseline_docling.json` → log error rõ "Run `python eval/run_docling.py` trước" + exit 1.

3. **Load snapshot:** `native = json.load(...)`, `docling = json.load(...)`.

4. **Heading recall logic** (CONTEXT mục "Implementation Tasks > Plan 05-02"):
   - Load `eval/dataset/headings.json` — dict `{filename: [heading_text, ...]}`.
   - Cho mỗi doc trong snapshot:
     - Lấy `doc_id` từ snapshot.
     - Query Postgres direct (psycopg, faster + tránh expose endpoint internal — CONTEXT Auto-mode log "Heading recall query: DB direct"):
       ```sql
       SELECT content FROM document_chunks WHERE document_id = %s
       ```
     - Cho mỗi heading vàng `h` trong `headings[filename]`:
       - Match nếu **có chunk nào chứa substring `h`** (case-insensitive, normalize whitespace).
     - `heading_recall = matched / total_headings`.

5. **Table preservation logic:**
   - Query Postgres:
     ```sql
     SELECT metadata FROM document_chunks WHERE document_id = %s
     ```
   - Đếm chunks có `metadata->>'is_table' = 'true'` (hoặc parse JSON nếu cần).
   - `tables_preserved` = count đó.
   - `tables_total` = số bảng vàng trong file gốc. Hardcode mapping ngắn ở đầu script (theo CONTEXT — Phase 1 đã label trong dataset, ví dụ T1-01 có 4 bảng, T3-02 có ma trận 4×7 = 1 bảng lớn, T1-02 có 0 bảng…). Fallback nếu không biết: dùng count `<table>` từ DOCX qua python-docx, hoặc ghi `null`.

6. **Compute delta per-document:** chunks_count diff, avg_chunk_tokens diff, heading_recall_pp (`(d - n) * 100`), tables_preserved diff.

7. **Compute summary:** average heading_recall toàn bộ + table preservation rate (sum_preserved / sum_total).

8. **Sinh `eval/extraction_compare.json`** đúng schema `<interfaces>`.

9. **Logging progress** mỗi doc (logger.info).

10. **Exit 0 thành công, exit 1 nếu pre-condition fail.**

LƯU Ý:
- 2 scanned PDF (`q09`, `q10`) ở mode native sẽ có `heading_recall=0` (vì `no text extracted`). Không cần special case — script tự nhiên trả 0.
- Match heading dùng `chunk.content.lower().replace(\\s+, " ")` chứa `heading.lower().replace(\\s+, " ")` (substring loose match — tránh false negative do whitespace/format).

LƯU Ý runtime: Script này cần `baseline_docling.json` + Postgres có data Docling đã ingest → **runtime defer cho user** (`runtime_smoke: deferred`). Executor commit code + verify import.
