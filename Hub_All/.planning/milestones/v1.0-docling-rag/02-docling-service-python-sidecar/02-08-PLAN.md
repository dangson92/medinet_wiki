```markdown
# docling-pipeline — Medinet Docling Sidecar

Service Python FastAPI sidecar bọc [Docling](https://github.com/DS4SD/docling) v2.91.0 — extract + chunk tài liệu đa định dạng cho RAG ingestion (M1).

**Trách nhiệm:**
- Nhận file binary qua `POST /v1/process`.
- Extract: PDF / DOCX / XLSX / PPTX / HTML / PNG / JPEG → `DoclingDocument` (1 luồng, OCR Tesseract `vie+eng` cho scanned PDF).
- Chunk: `HybridChunker` với tokenizer `cl100k_base` (default cho OpenAI text-embedding-3-*).
- Trả JSON đúng schema DSVC-02: `{request_id, doc_meta, chunks: [...]}`.

**KHÔNG làm:** embedding, ChromaDB upsert, augmenter Q&A — backend Go (`backend/internal/`) vẫn xử lý.

## Cài đặt + chạy

### Local (dev nhanh, không Docker)

```bash
cd docling-pipeline

# 1. Tạo venv + install
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -e .[dev]

# 2. Cần Tesseract OCR ngoài pip — cài qua OS
#    Ubuntu/Debian:
sudo apt-get install -y tesseract-ocr tesseract-ocr-vie tesseract-ocr-eng
#    macOS:
brew install tesseract tesseract-lang
#    Windows: tải từ https://github.com/UB-Mannheim/tesseract/wiki, thêm PATH

# 3. Copy env
cp .env.example .env

# 4. Run
uvicorn docling_pipeline.main:app --host 0.0.0.0 --port 8001 --workers 1
```

Lần đầu start sẽ tải model Docling (~vài GB từ HuggingFace) — 30-60s. `/readyz` trả 503 nếu library không cài được; trả 200 sau khi warm xong (B5).

### Docker (khuyến nghị)

```bash
# Từ ROOT repo Hub_All/
cd ..

# Build (chậm 5-10 phút lần đầu)
docker compose build docling-pipeline

# Start cùng infra (postgres + redis + chroma + docling)
docker compose up -d
docker compose logs -f docling-pipeline      # đợi log "models_warmed" (state=ready)
```

**Note:** Service KHÔNG expose port 8001 ra host (security — DSVC-05). Để test thủ công từ host, override tạm trong `docker-compose.override.yml`:

```yaml
services:
  docling-pipeline:
    ports:
      - "8001:8001"
```

## Schema response `POST /v1/process` (DSVC-02)

```json
{
  "request_id": "uuid-từ-X-Request-Id",
  "doc_meta": {
    "filename": "doc.pdf",
    "file_type": "pdf",
    "page_count": 12,
    "language_detected": null,
    "ocr_used": false
  },
  "chunks": [
    {
      "chunk_index": 0,
      "text": "...markdown...",
      "headers": ["PHẦN 01", "1.1 ..."],
      "caption": null,
      "page_start": 1,
      "page_end": 1,
      "is_table": false,
      "table_html": null,
      "bbox": [50.0, 60.0, 500.0, 700.0],
      "token_count": 234
    }
  ]
}
```

Khi `is_table=true`: `text` chứa Markdown table flatten, `table_html` chứa HTML đầy đủ với `<thead>/<tbody>/colspan/rowspan` (EXTRACT-03).

Khi figure có caption: text bắt đầu bằng `![<caption>](#fig-N)` (EXTRACT-04).

## Troubleshooting

**`/readyz` luôn 503 với `reason: docling_library_unavailable`:** Docling library không import được trong container (B5 case a). Check `pip list | grep docling` trong container; nếu thiếu → rebuild image.

**`/readyz` 200 nhưng request đầu rất chậm:** B5 case b — warm dummy fail nhưng library OK. Bình thường, request đầu sẽ trigger model load (~30-60s).

**Container build chậm:** Bình thường lần đầu (5-10 phút). Lần sau cache layer apt + pip; chỉ rebuild khi đổi pyproject.toml.

**Port 5432/8000 conflict khi `docker compose up`:** Postgres/ChromaDB local đang chạy trên cùng port. Stop service local trước, hoặc đổi port trong `.env` root (`POSTGRES_PORT=5433`).

**OCR test fail tiếng Việt:** Tesseract không có language pack `vie`. Verify: `tesseract --list-langs | grep vie`. Cài: `apt-get install tesseract-ocr-vie`.

**`docling_models/` chiếm dung lượng lớn:** Đúng — HuggingFace cache vài GB. Volume mount để khỏi re-download.
