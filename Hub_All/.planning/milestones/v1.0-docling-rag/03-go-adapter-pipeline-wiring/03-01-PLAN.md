**File 1 — `backend/internal/pkg/requestid/requestid.go`** (MỚI):

```go
// Package requestid cung cấp key chuẩn + helper trích xuất request_id
// từ Gin context hoặc context.Context — dùng cho cross-service tracing
// (Go ↔ docling-pipeline qua header X-Request-Id).
package requestid

import (
    "context"

    "github.com/gin-gonic/gin"
)

// Key là khóa được middleware.RequestID lưu vào Gin context.
// Cũng dùng làm context.Value key (kiểu string đơn giản — không cần
// custom type vì chỉ middleware này set).
const Key = "request_id"

// HeaderName là tên HTTP header dùng cross-service.
const HeaderName = "X-Request-Id"

// From trả request_id từ ctx. Hỗ trợ cả `*gin.Context` và `context.Context`
// (vì Gin context implement cả 2 interface — nhưng pipeline.go truyền
// xuống worker là context.Context thuần, cần fallback). Trả "" nếu không có.
func From(ctx context.Context) string {
    if c, ok := ctx.(*gin.Context); ok {
        if v, exists := c.Get(Key); exists {
            if s, ok := v.(string); ok {
                return s
            }
        }
        return ""
    }
    if v := ctx.Value(Key); v != nil {
        if s, ok := v.(string); ok {
            return s
        }
    }
    return ""
}

// With gắn request_id vào context.Context (dùng khi worker nhận job
// từ queue và muốn propagate xuống pipeline).
func With(ctx context.Context, rid string) context.Context {
    return context.WithValue(ctx, ctxKey{}, rid)
}

type ctxKey struct{}
```

LƯU Ý: Cập nhật `From` để check cả `ctxKey{}` lẫn string `Key` — vì Gin set qua `c.Set(Key, ...)` (kiểu string), còn `With` dùng `ctxKey{}` để tránh collision. Final implementation:

```go
func From(ctx context.Context) string {
    if c, ok := ctx.(*gin.Context); ok {
        if v, exists := c.Get(Key); exists {
            if s, ok := v.(string); ok { return s }
        }
        return ""
    }
    if v, ok := ctx.Value(ctxKey{}).(string); ok && v != "" {
        return v
    }
    if v, ok := ctx.Value(Key).(string); ok {
        return v
    }
    return ""
}
```

**File 2 — `backend/internal/middleware/request_id.go`** (MỚI):

```go
package middleware

import (
    "github.com/gin-gonic/gin"
    "github.com/google/uuid"

    "github.com/medinet/hub-all-backend/internal/pkg/requestid"
)

// RequestID đảm bảo mọi request có header X-Request-Id (auto-gen UUID v4
// nếu thiếu) và lưu vào Gin context dưới key requestid.Key. Cũng echo
// header trở lại response để client/proxy log được.
//
// Đặt sớm trong middleware chain (ngay sau Recovery, trước SecurityHeaders)
// để mọi log/error downstream có request_id.
func RequestID() gin.HandlerFunc {
    return func(c *gin.Context) {
        rid := c.GetHeader(requestid.HeaderName)
        if rid == "" {
            rid = uuid.NewString()
        }
        c.Set(requestid.Key, rid)
        c.Header(requestid.HeaderName, rid)
        c.Next()
    }
}
```

LƯU Ý:
- `github.com/google/uuid` đã có trong `go.mod` (đã dùng ở `worker/manager.go`, `pipeline.go`) — KHÔNG cần `go get`.
- KHÔNG validate format UUID của header client gửi — accept as-is (Gateway/proxy có thể inject format khác như `traceparent`).
