  KHÔNG thêm migration cột mới (REQUIREMENTS ASK-05 không yêu cầu; Phase 10 HARD-01 mới đụng observability).
- D-07-02-B — `GET /api/usage` raw JSON KHÔNG envelope? KHÔNG — usage là endpoint port
  thường, dùng envelope `{success,data,error,meta}` như search/auth (chỉ `/api/rag-config`
  cố tình raw — D6). Frontend `request<TokenUsageAPI[]>` đọc qua `APIResponse` wrapper.
- D-07-02-C — `realtime` (window 60 phút, group theo phút) M2 làm best-effort từ cùng
  bảng `usage_events` `WHERE created_at > NOW() - INTERVAL '60 minutes'` — không stream.

  → `GET /api/usage` PHẢI nhận query param optional `group_by`. Khi `group_by` có giá trị

  và trả về aggregate y hệt `/api/usage/stats` (data = `UsageStats.model_dump()`). Khi
  `group_by` không truyền → trả list `UsageEventResponse[]` như cũ. GIỮ luôn endpoint
  riêng `/api/usage/stats` (frontend D6 dùng) — `group_by` chỉ là cửa thứ hai để URL
  literal SC5 hoạt động. Cả 2 cùng gọi `aggregate_usage()` → kết quả nhất quán.
