# Plan 03: CocoIndex flow medinet_wiki_ingest + lifespan FlowLiveUpdater

## Objective

Định nghĩa cocoindex flow đầy đủ (INGEST-01..03) bằng cách wrap services Plan 04-02 thành transform chain. Khởi động flow qua FastAPI lifespan để cocoindex tự pick up document mới qua Postgres LISTEN/NOTIFY khi router upload (Plan 04-04) INSERT vào bảng `documents`.

**Tại sao plan riêng:**
- Plan 04-02 tách services khỏi cocoindex để test unit nhanh. Plan 04-03 wire vào cocoindex `@cocoindex.flow_def` + `cocoindex.op.function` decorator.
- Plan 04-04 (router upload) chỉ INSERT row `documents` rồi return 202; cocoindex trigger qua NOTIFY là responsibility của Plan 04-03.
- Tách lifespan wire (Plan 04-03) khỏi router (Plan 04-04) — lifespan có complexity riêng (FlowLiveUpdater lifecycle + abort timeout) cần atomic commit.

**Critical insight cocoindex 1.0.3 API (revision 1 — BLOCKER #1 + #2):**

- `@cocoindex.flow_def(name="medinet_wiki_ingest")` decorator + signature `def flow(flow_builder, data_scope)`.
- `data_scope["documents"] = flow_builder.add_source(cocoindex.sources.Postgres(...))`.
- `@cocoindex.op.function()` PHẢI return TYPED struct (`dataclass` / `TypedDict` / `NamedTuple` / Pydantic BaseModel). KHÔNG return `dict[str, Any]` hay `list[dict[str, Any]]` — cocoindex 1.0.3 type inspector reject (BLOCKER #1).
- `chunk.export("chunks", primary_key_fields=["id"], vector_indexes=[cocoindex.VectorIndexDef(field_name="vector", metric=cocoindex.VectorSimilarityMetric.COSINE_SIM)])` — param **CHÍNH XÁC** là `field_name` keyword (verified cocoindex 1.0.3 source — xem STACK.md + ARCHITECTURE.md line 311).
- Chunk row PHẢI wire `hub_id`, `document_id`, `content_hash`, `metadata`, `vector` TRƯỚC `chunk.export(...)` — schema chunks migration 0001 yêu cầu `hub_id NOT NULL FK CASCADE` + `content_hash BYTEA NOT NULL`. Thiếu → constraint violation, flow fail mọi document (BLOCKER #2).
- Live update: `updater = cocoindex.FlowLiveUpdater(flow_obj); updater.start()` + `updater.abort()` ở shutdown. `updater.start()` sync method → wrap `asyncio.to_thread` trong lifespan async context (WARNING #1).

**R4 mitigation (BLOCKER #3 — strategy A router-side synchronous early-detect):**
- Logic detect scanned PDF + set `status='failed_unsupported'` được **router/service Plan 04-04** xử lý SYNCHRONOUSLY tại upload time (call `extract_text` Plan 04-02 trực tiếp; nếu `is_scanned=True` → 415 envelope + INSERT row status='failed_unsupported' + KHÔNG pg_notify).
- Plan 04-03 flow `_extract_op` vẫn return `ExtractResult.is_scanned: bool` (defensive — nếu router miss detection vì lý do nào đó), NHƯNG **KHÔNG cần wire `status='failed_unsupported'` ở flow layer** — vì router đã loại scanned PDF khỏi đường vào flow trước khi INSERT row pending. @cocoindex.op.function()

```python
Source: Postgres `documents` table với LISTEN/NOTIFY channel `documents_notify`.
        Trigger khi Plan 04-04 router INSERT row mới → cocoindex auto pick up.

Transform chain:
    documents row → extract_text (Plan 04-02 file_extract) → ExtractResult typed
                  → chunk_vietnamese (Plan 04-02 vn_chunker) → list[ChunkPayload] typed
                  → embed_text (Plan 04-02 embedder, dim 1536)
                  → hash content (sha256 → bytes)
                  → chunks table với hub_id + document_id + content_hash wire đủ

Target: `chunks` table với `vector vector(1536)` + HNSW cosine index.
        Cocoindex stable chunk_id qua re-index (D-1, D-2 — citation preserve).

Mitigations:
- BLOCKER #1: op function return TYPED dataclass (ExtractResult, ChunkPayload) —
  cocoindex 1.0.3 KHÔNG hỗ trợ `dict[str, Any]` return.
- BLOCKER #2: chunk row wire ĐẦY ĐỦ schema (hub_id, document_id, content_hash,
  metadata, vector) — schema chunks migration 0001 yêu cầu hub_id NOT NULL +
  content_hash NOT NULL; thiếu → constraint violation flow fail mọi document.
- BLOCKER #3 (R4 scanned PDF): logic set status='failed_unsupported' chuyển sang
  router/service Plan 04-04 synchronous early-detect TRƯỚC INSERT documents.
  Flow Plan 04-03 chỉ chạy với DOCX/TXT/MD/PDF text-only. _extract_op vẫn return
  is_scanned defensive nhưng KHÔNG cần wire status ở flow layer.
- WARNING #3: VectorIndexDef param `field_name="vector"` keyword (verified
  cocoindex 1.0.3 source — ARCHITECTURE.md line 311 + STACK.md).
- R5 (naming): flow name="medinet_wiki_ingest" snake_case (P2).

"""
from __future__ import annotations

import hashlib
from dataclasses import dataclass
from pathlib import Path

import cocoindex

from app.services.embedder import embed_text
from app.services.file_extract import extract_text
from app.services.vn_chunker import chunk_vietnamese

# === Typed structs cho cocoindex op return (BLOCKER #1) ===


@dataclass
class ExtractResult:
    """Output _extract_op — cocoindex 1.0.3 yêu cầu typed struct return.

    Fields:
        text: nội dung text extracted.
        is_scanned: True nếu pypdf detect > 80% pages < 30 chars (defensive —
            router Plan 04-04 đã early-detect; flow KHÔNG cần xử lý nữa).
        pages: số page (DOCX/TXT/MD = 1).
        format: docx | txt | md | pdf.
    """

    text: str
    is_scanned: bool
    pages: int
    format: str


@dataclass
class ChunkPayload:
    """Output _chunk_op (per chunk) — cocoindex 1.0.3 yêu cầu typed struct.

    Plan 04-02 vn_chunker ChunkDraft → map qua ChunkPayload cho cocoindex flow.
    """

    content: str
    heading_path: str | None
    page_start: int
    page_end: int


# === Transform ops — wrap services Plan 04-02 (typed return) ===


@cocoindex.op.function()
def _extract_op(file_path: str) -> ExtractResult:
    """Extract text + scanned-detect + meta. Return typed ExtractResult."""
    text, is_scanned, meta = extract_text(Path(file_path))
    return ExtractResult(
        text=text,
        is_scanned=is_scanned,
        pages=int(meta.get("pages", 1)),
        format=str(meta.get("format", "unknown")),
    )


@cocoindex.op.function()
def _chunk_op(text: str) -> list[ChunkPayload]:
    """Chunk VN text. Return list[ChunkPayload] typed."""
    drafts = chunk_vietnamese(text)
    return [
        ChunkPayload(
            content=d.content,
            heading_path=d.heading_path,
            page_start=d.page_start,
            page_end=d.page_end,
        )
        for d in drafts
    ]


@cocoindex.op.function()
async def _embed_op(content: str) -> list[float]:
    """Embed chunk content qua LiteLLM (dim 1536)."""
    return await embed_text(content)


@cocoindex.op.function()
def _hash_op(content: str) -> bytes:
    """SHA-256 stable hash → BYTEA cho chunks.content_hash NOT NULL (BLOCKER #2)."""
    return hashlib.sha256(content.encode("utf-8")).digest()


# === Flow def ===


@cocoindex.flow_def(name="medinet_wiki_ingest")  # R5 snake_case
def medinet_wiki_ingest_flow(  # noqa: C901 — flow DSL có nhiều with-blocks
    flow_builder: cocoindex.FlowBuilder,
    data_scope: cocoindex.DataScope,
) -> None:
    """INGEST-01..03 — Postgres LISTEN/NOTIFY → extract → chunk → embed → chunks.

    Chunk row wire ĐẦY ĐỦ schema chunks (BLOCKER #2):
        hub_id, document_id, content_hash, heading_path, page_start, page_end,
        metadata, vector. (id + created_at do Postgres server-side default.)
    """
    data_scope["documents"] = flow_builder.add_source(
        cocoindex.sources.Postgres(
            table_name="documents",
            ordinal_column="updated_at",
            notification=cocoindex.sources.PostgresNotification(
                channel="documents_notify",
            ),
        ),
    )

    with data_scope["documents"].row() as doc:
        # Transform 1: extract text + scanned detect.
        doc["extracted"] = doc["file_path"].transform(_extract_op)

        # Transform 2: chunk VN. Logic skip-if-scanned thuộc router Plan 04-04
        # synchronous early-detect (BLOCKER #3 strategy A). Flow Plan 04-03 chỉ
        # chạy với DOCX/TXT/MD/PDF text-only — _extract_op trả is_scanned=False.
        doc["chunks"] = doc["extracted"]["text"].transform(_chunk_op)

        with doc["chunks"].row() as chunk:
            # === BLOCKER #2: wire ĐẦY ĐỦ schema chunks ===
            chunk["hub_id"] = doc["hub_id"]
            chunk["document_id"] = doc["id"]
            chunk["content_hash"] = chunk["content"].transform(_hash_op)
            # heading_path / page_start / page_end vừa wire direct (cho query
            # đơn giản) vừa nested trong metadata JSONB (cho admin debug).
            chunk["metadata"] = {
                "heading_path": chunk["heading_path"],
                "page_start": chunk["page_start"],
                "page_end": chunk["page_end"],
            }
            chunk["vector"] = chunk["content"].transform(_embed_op)

            # === Export với VectorIndexDef field_name keyword (WARNING #3) ===
            chunk.export(
                "chunks",
                cocoindex.targets.Postgres(table_name="chunks"),
                primary_key_fields=["id"],
                vector_indexes=[
                    cocoindex.VectorIndexDef(
                        field_name="vector",
                        metric=cocoindex.VectorSimilarityMetric.COSINE_SIM,
                    )
                ],
            )


__all__ = [
    "ChunkPayload",
    "ExtractResult",
    "medinet_wiki_ingest_flow",
]
```

```python
def stop_flow_live_updater(updater: object) -> None:
    """Gracefully abort FlowLiveUpdater (shutdown lifespan).

    Args:
        updater: cocoindex.FlowLiveUpdater instance từ start_flow_live_updater().
    """
    if updater is None:
        return
    try:
        # Cocoindex 1.0.3 method name có thể là .abort() hoặc .stop() — try cả 2.
        if hasattr(updater, "abort"):
            updater.abort()
        elif hasattr(updater, "stop"):
            updater.stop()
        else:
            logger.warning("cocoindex_flow_updater_no_abort_method: %r", updater)
            return
        logger.info("cocoindex_flow_live_updater_stopped")
    except Exception as e:  # noqa: BLE001
```
