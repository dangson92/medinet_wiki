## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP client → MCP Service (Streamable HTTP) | Client gửi `X-API-Key` qua header MCP. |
| MCP Service → API Service (HTTP) | Tool forward key xuống API Service; API Service verify. |
| MCP tool error → MCP client | Lỗi nội bộ không được rò ra client. |
