<objective>
Mở auth API Service để MCP Service (gọi qua HTTP, forward `X-API-Key` của client) sử dụng được toàn bộ 4 endpoint mà 3 tool MCP cần: `GET /api/hubs` (tool `list_hubs`), `POST /api/search` + `/api/search/cross-hub` (tool `search_wiki`), `POST /api/ask` + `/api/search/answer` (tool `ask_wiki`).

Hiện trạng (gốc BLOCKER revision):
- `GET /api/hubs` là admin-only (`require_role("admin")`) → API key non-admin không gọi được.
- `POST /api/search`, `/api/search/cross-hub`, `/api/ask`, `/api/search/answer` dùng `Depends(get_current_user_with_hubs)` → chain sang `get_current_user` (`dependencies.py:84-89`) CHỈ đọc Bearer JWT qua `oauth2_scheme`, KHÔNG đọc header `X-API-Key`. MCP Service forward X-API-Key (không có JWT) → token rỗng → 401 `MISSING_AUTHORIZATION`. Hệ quả: `search_wiki`/`ask_wiki` LUÔN trả `ToolError MCP_UNAUTHORIZED` → SC2/SC3/SC4 không đạt.

Lưu ý kỹ thuật then chốt: search/ask cần `UserWithHubs` (user + `hub_ids` load từ `user_hubs`) cho hub isolation API-side, KHÔNG phải `User` thuần. `get_api_key_or_jwt` hiện có chỉ trả `User` → không đủ. Plan này thêm dependency mới `get_api_key_or_jwt_with_hubs` (verify X-API-Key HOặC JWT → load `hub_ids` → trả `UserWithHubs`) — analog cặp `get_current_user` / `get_current_user_with_hubs`.

Plan này: (1) thêm `get_api_key_or_jwt_with_hubs` vào `dependencies.py`; (2) `GET /api/hubs` → `get_api_key_or_jwt` + scope theo role; (3) đổi 4 endpoint search/ask sang `get_api_key_or_jwt_with_hubs`; (4) test verify cả X-API-Key path + hub scope cho admin/non-admin. Các endpoint hubs mutate (POST/PUT/PATCH/stats) GIỮ NGUYÊN admin-only.

Mục đích: Điều kiện tiên quyết cho Plan 03 (3 tool MCP gọi API qua HTTP). Đáp ứng SC2 (3 tool gọi API forward X-API-Key), SC3 (auth fail → 401 → ToolError; hub isolation enforce API-side), SC4 (usage_events vẫn ghi sau /api/ask qua X-API-Key).
Output: `get_api_key_or_jwt_with_hubs` trong `dependencies.py`; 4 endpoint search/ask + `GET /api/hubs` nhận X-API-Key; `HubService.list_for_hubs`; test unit verify cả admin/non-admin + X-API-Key path.
