## Điểm chính

- **Package độc lập:** `mcp_service/pyproject.toml` chỉ khai báo `mcp`, `httpx`, `pydantic`, `pydantic-settings` — `grep -rE "fastapi|cocoindex|asyncpg|import app\."` trên `mcp_app/` trả rỗng (CLEAN). Build backend hatchling, ruff/mypy/pytest config nhất quán với `api/pyproject.toml`.
- **config.py:** `Settings(BaseSettings)` env prefix `MCP_`. `field_validator` cho `api_base_url` strip trailing slash, ép scheme http/https, bắt buộc có host. `get_settings()` cache singleton.
- **schemas.py:** Port nguyên 6 model từ `api/app/mcp/schemas.py` — giữ field + type chính xác để MCP tool trả cùng shape.
- **api_client.py:** `ApiClient` dùng `httpx.AsyncClient` với timeout (chống request treo — T-08.2-01-D), hỗ trợ `async with`. `_request` unwrap envelope `{success,data,error}`, map 401→`ApiUnauthorizedError`, 403→`ApiForbiddenError`, 400→`ApiBadRequestError`, 5xx/LLM_FAILED→`ApiServerError`. Network error → `ApiServerError`.
- **Threat mitigations áp dụng:** T-08.2-01-I (log chỉ `method`/`path`/`status_code`/`error_code` — không leak `api_key`/header), T-08.2-01-T (validate scheme base URL), T-08.2-01-D (timeout httpx).
