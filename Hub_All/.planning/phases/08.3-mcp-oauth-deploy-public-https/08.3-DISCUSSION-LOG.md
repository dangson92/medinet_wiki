## Mapping danh tính → hub scope

### Q1: MCP Service xác thực xuống API Service thế nào?

| Option | Description | Selected |
|--------|-------------|----------|
| Forward JWT theo từng user | OAuth login → MCP login API Service như user đó → JWT, forward mỗi call | ✓ |
| API key theo từng user | Mỗi user phải có sẵn api_keys row; MCP tra key | |
| Một API key dùng chung | Mọi tool call dùng chung 1 service key, cùng hub scope | |

**User's choice:** Forward JWT theo từng user
**Notes:** API `get_api_key_or_jwt` đã chấp nhận JWT; không cần pre-provision; hub scope live mỗi call.
