QUYẾT ĐỊNH PLANNER (REQUIREMENTS/ROADMAP là source of truth — gray area resolve):

- D-07-03-A — Bản đồ model → dimension. ROADMAP SC4 nêu `gemini-embedding-001@1536` và
  cross-dim `1536↔3072`. Quy ước: model name chứa hậu tố `@<dim>` (vd `...@3072`) → dim =
  số đó; nếu KHÔNG có hậu tố → mặc định 1536 (M2 pin 1536). Hàm `_embedding_dim_of(model)`
  parse: regex `@(\d+)$` → int, fallback `1536`. Đây là quy ước M2 — frontend Settings gửi
  model name; tài liệu hoá rõ trong docstring.
- D-07-03-B — Cost preview công thức (echo cho UI, không cần chính xác tuyệt đối):
  - `n_chunks` = `SELECT count(*) FROM chunks` (toàn corpus — re-embed ảnh hưởng tất cả).
  - `est_cost_usd` = `n_chunks * COST_PER_CHUNK_USD` với hằng `COST_PER_CHUNK_USD = 0.000013`
    (≈ text-embedding-3-small $0.02/1M token, ~650 token/chunk trung bình). Làm tròn 2 chữ số.
  - `est_minutes` = `ceil(n_chunks / CHUNKS_PER_MINUTE)` với `CHUNKS_PER_MINUTE = 450`
    (~7-8 embed/s qua LiteLLM). Tối thiểu 1.
  - Message tiếng Việt CHÍNH XÁC format: `f"re-embed {n} chunks, est ${cost:.2f}, est {m} phút"`.
    BẮT BUỘC dùng format `:.2f` để cost LUÔN có 2 chữ số thập phân — ROADMAP SC4 nêu
    "$0.34" 2 chữ số; `round(n*rate, 2)` cho 0.30 sẽ in "0.3" (Python strip trailing zero)
    → fail strict verbatim check. `f"${cost:.2f}"` cho ra "$0.30" / "$0.34" ổn định.
    Message output PHẢI khớp regex `est \$\d+\.\d{2},`.
- D-07-03-C — Khi swap embedding WITHIN dim 1536: KHÔNG tự động re-embed (M2 không có
  trigger re-embed hàng loạt — đó là v4.0). Chỉ trả WARNING + cost preview để admin biết.
  Config MỚI VẪN được persist (provider đổi áp dụng cho document upload SAU đó). Đây là
  hành vi đúng theo R7 ("allowed với WARNING modal").
- D-07-03-D — Router contract: `/api/rag-config` GIỮ raw JSON KHÔNG envelope (D6). Cross-dim
  refuse: `update_config()` trả `str` → router hiện đã map `JSONResponse(status_code=400,
  content={"error": result})`. WARNING cost preview: trả trong dict success, thêm key
  `warning` + `cost_preview`. KHÔNG đổi chữ ký router — chỉ mở rộng nội dung dict.
  Router `rag_config.py` KHÔNG nằm trong `files_modified` — plan này chỉ đụng service +
  schema + test; router đã sẵn sàng map cả 2 nhánh.
