EOSQL

3. **Bước (1/4)** loop tạo 3 DB hub con `medinet_hub_<hub>`. Pattern:
   ```bash
   for hub in "${HUBS[@]}"; do
       db_name="medinet_hub_${hub}"
       echo "[init-db] (1/4) Tạo database ${db_name}..."
       # Idempotent: dùng SELECT pg_database WHERE datname rồi conditional CREATE
       # vì Postgres KHÔNG hỗ trợ `CREATE DATABASE IF NOT EXISTS`.
       exists=$(psql -tA -v ON_ERROR_STOP=1 --username "${POSTGRES_USER}" --dbname "postgres" \
           -c "SELECT 1 FROM pg_database WHERE datname='${db_name}'")
       if [ "$exists" != "1" ]; then
           psql -v ON_ERROR_STOP=1 --username "${POSTGRES_USER}" --dbname "postgres" \
               -c "CREATE DATABASE ${db_name} OWNER ${POSTGRES_USER};"
       else
           echo "[init-db] (1/4) ${db_name} đã tồn tại, skip."
       fi
   done
   ```

4. **Bước (2/4)** giữ tạo `medinet_cocoindex` (M2 carry forward) — cocoindex internal state DB. Cùng pattern conditional CREATE để idempotent. Comment: R5/P7 — cocoindex schema cố định trong DB này.

5. **Bước (3/4)** loop enable extension vector trên 5 DB: `medinet_central` + 3 hub con + `medinet_cocoindex`. Pattern:
   ```bash
   ALL_DBS=("medinet_central" "medinet_cocoindex")
   for hub in "${HUBS[@]}"; do ALL_DBS+=("medinet_hub_${hub}"); done
   for db in "${ALL_DBS[@]}"; do
       echo "[init-db] (3/4) Enable extension vector trên ${db}..."
       psql -v ON_ERROR_STOP=1 --username "${POSTGRES_USER}" --dbname "${db}" <<-EOSQL
           CREATE EXTENSION IF NOT EXISTS vector;
EOSQL
   done
   ```

6. **Bước (4/4)** VERIFY HNSW 1536-dim build trên TẤT CẢ 4 DB nghiệp vụ (`medinet_central` + 3 hub con — KHÔNG verify `medinet_cocoindex` vì cocoindex internal). Pattern:
   ```bash
   VERIFY_DBS=("medinet_central")
   for hub in "${HUBS[@]}"; do VERIFY_DBS+=("medinet_hub_${hub}"); done
   for db in "${VERIFY_DBS[@]}"; do
       echo "[init-db] (4/4) VERIFY HNSW vector(1536) build OK trên ${db}..."
       psql -v ON_ERROR_STOP=1 --username "${POSTGRES_USER}" --dbname "${db}" <<-EOSQL
           DROP TABLE IF EXISTS _hnsw_dim_check;
           CREATE TABLE _hnsw_dim_check (id serial primary key, v vector(1536));
           CREATE INDEX _hnsw_dim_check_idx ON _hnsw_dim_check USING hnsw (v vector_cosine_ops);
           DROP TABLE _hnsw_dim_check;
EOSQL
   done
   ```

7. Kết thúc bằng `echo "[init-db] DONE — 4 nghiệp vụ DB + cocoindex created, vector ext enabled, HNSW 1536-dim verified."`

Lý do dùng `SELECT pg_database WHERE datname` thay vì `CREATE DATABASE IF NOT EXISTS`: Postgres KHÔNG hỗ trợ syntax `IF NOT EXISTS` cho `CREATE DATABASE` — chỉ work cho EXTENSION/TABLE/INDEX. Conditional check via `psql -tA -c "SELECT ..."` là pattern idempotent chuẩn.

Encoding: pure UTF-8, LF line endings (script chạy trong Postgres container Linux). KHÔNG dùng tiếng Việt có dấu trong echo string (locale container có thể không hỗ trợ — echo ASCII OK).

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Docker host → Postgres container | POSTGRES_PASSWORD env truyền từ host shell qua compose; init script chạy với quyền superuser POSTGRES_USER lúc cluster lần đầu |
| Postgres cluster → init script | init-db.sh thực thi với privilege POSTGRES_USER (superuser) tại entrypoint — có thể CREATE DATABASE bất kỳ |
| Init script → từng DB | psql heredoc connect mỗi DB riêng để CREATE EXTENSION + verify HNSW; không cross-DB privilege escalation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01-01 | Tampering | `init-db.sh` script content | mitigate | Mount ro (`:ro`) đã có ở docker-compose.yml volume; bash script idempotent guard `SELECT pg_database WHERE datname` chống double-CREATE; file owner tự repo (git-tracked). |
| T-01-01-02 | Elevation of Privilege | POSTGRES_USER `medinet` superuser ở cluster init | accept | Cluster init chỉ chạy 1 lần lúc volume empty; superuser cần thiết để CREATE DATABASE (Postgres không hỗ trợ delegate); production deploy sẽ phải set POSTGRES_PASSWORD strong qua env (không default `medinet_dev_pwd`). Defer hardening v4.0. |
| T-01-01-03 | Denial of Service | HNSW verify 1536-dim DROP TABLE IF EXISTS race | mitigate | `DROP TABLE IF EXISTS` trước CREATE để re-run init không fail; 4 DB verify tuần tự ~5s mỗi DB — OK với healthcheck retries=10 interval=5s timeout=3s. |
| T-01-01-04 | Information Disclosure | psql log echo "medinet_hub_<name>" lên stdout container | accept | Tên DB không phải PII / secret; log container chỉ visible cho admin docker host. |
