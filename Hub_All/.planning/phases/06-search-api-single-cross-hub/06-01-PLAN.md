Helper module-level (dùng chung cho 06-02):

`_to_pgvector_literal(vec: list[float]) -> str` — trả `"[" + ",".join(repr(float(x)) for x in vec) + "]"`. SQL cast `$1::vector`.

`_truncate_snippet(content: str, limit: int = SNIPPET_MAX_CHARS) -> str` — cắt content tối đa `limit` ký tự theo word boundary: nếu `len(content) <= limit` trả nguyên; ngược lại cắt `content[:limit]`, rsplit theo space lấy phần trước space cuối, append `"…"`.

`_cache_key(query: str, hub_ids: list[str], top_k: int, min_score: float | None) -> str` — trả `"search:" + hashlib.sha256(f"{query}|{sorted(hub_ids)}|{top_k}|{min_score}".encode()).hexdigest()`.

`_resolve_top_k(raw: int | None) -> int` — `raw` None → DEFAULT_TOP_K; clamp vào `[1, MAX_TOP_K]`.

`intersect_hubs(requested: list[str] | None, user_hub_ids: list[str], role: str) -> list[str]` — DEFENSE IN DEPTH LỚP 1 (D-07):
  - role == "admin": nếu `requested` rỗng/None → trả `[]` đặc biệt nghĩa "mọi hub" — DÙNG sentinel: trả `requested or []` cho admin và caller hiểu `[]` = all-hubs. ĐỂ ĐƠN GIẢN: admin + requested None → trả `[]`; admin + requested có giá trị → trả `list(requested)`. (admin search mọi hub khi không filter.)
  - role editor/viewer: `base = set(requested) if requested else set(user_hub_ids)`; trả `sorted(base & set(user_hub_ids))`. Hub user không quyền bị LOẠI — KHÔNG raise (D-07 silent-drop).

`_row_to_item(row: Any) -> SearchResultItem` — map 1 asyncpg Record → SearchResultItem:
  - `id=str(row["id"])`, `hub_id=str(row["hub_id"])`, `hub_name=row["hub_name"]`
  - `title=row["filename"]` (D-10 documents.filename → title)
  - `content=row["content"]`, `snippet=_truncate_snippet(row["content"])`
  - `category=None`, `tags=[]`, `source="document"` (D-10 hằng)
  - `score=float(row["score"])`, `raw_similarity=float(row["score"])` (D-10 — M2 chưa rerank)
  - `updated_at=row["updated_at"].isoformat() if row["updated_at"] else None`

Method `async def _run_vector_query(self, *, query_vector: list[float], hub_ids: list[str], top_k: int, all_hubs: bool) -> list[Any]` — raw SQL với HNSW tuning (D-08, D-09):
```python
vec_literal = _to_pgvector_literal(query_vector)
async with self.pool.acquire() as conn:
    async with conn.transaction():
        await conn.execute("SET LOCAL hnsw.ef_search = 200")
        await conn.execute("SET LOCAL hnsw.iterative_scan = relaxed_order")
        await conn.execute("SET LOCAL hnsw.max_scan_tuples = 20000")
        if all_hubs:
            sql = (
                "SELECT c.id, c.document_id, c.hub_id, c.content, c.heading_path, "
                "c.metadata, d.filename, d.updated_at, h.name AS hub_name, "
                "1 - (c.vector <=> $1::vector) AS score "
                "FROM chunks c "
                "JOIN documents d ON d.id = c.document_id "
                "JOIN hubs h ON h.id = c.hub_id "
                "WHERE c.vector IS NOT NULL "
                "ORDER BY c.vector <=> $1::vector LIMIT $2"
            )
            return await conn.fetch(sql, vec_literal, top_k)
        sql = (
            "SELECT c.id, c.document_id, c.hub_id, c.content, c.heading_path, "
            "c.metadata, d.filename, d.updated_at, h.name AS hub_name, "
            "1 - (c.vector <=> $1::vector) AS score "
            "FROM chunks c "
            "JOIN documents d ON d.id = c.document_id "
            "JOIN hubs h ON h.id = c.hub_id "
            "WHERE c.hub_id = ANY($2::uuid[]) AND c.vector IS NOT NULL "
            "ORDER BY c.vector <=> $1::vector LIMIT $3"
        )
        return await conn.fetch(sql, vec_literal, hub_ids, top_k)
```
LƯU Ý: `all_hubs=True` CHỈ khi role admin + không filter hub_ids. Editor/viewer LUÔN `all_hubs=False` với `hub_ids` đã intersect — nếu `hub_ids` rỗng (user chưa assign hub) thì `= ANY(ARRAY[]::uuid[])` trả 0 row (đúng — KHÔNG leak). Caller phải check `hub_ids` rỗng + non-admin → trả results [] sớm, KHÔNG query.

  4. `is_admin_all = user.user.role == "admin" and not body.hub_ids`.
  5. Nếu KHÔNG `is_admin_all` và `hub_ids` rỗng → trả ngay `SearchResponse(results=[], total_hubs_searched=0, query_time_ms=int((time.perf_counter()-t0)*1000), cache_hit=False).model_dump(mode="json")` (user chưa assign hub — empty result hợp lệ, Specifics).
  6. Cache đọc (SEARCH-04, D-11): `cache_key = _cache_key(body.query, hub_ids, top_k, body.min_score)`. Nếu `self.redis is not None`: try `cached = await self.redis.get(cache_key)`; nếu `cached`: `result = json.loads(cached); result["cache_hit"] = True; return result`.
