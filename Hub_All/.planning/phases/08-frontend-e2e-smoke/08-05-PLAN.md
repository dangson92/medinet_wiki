<objective>
Đóng gap blocker SC5 (Phase 8 / Test 1 UAT): `docker compose up` → container `python-api`
crash ngay startup vì cocoindex `Environment.__init__` ném
`RuntimeError: Permission denied (os error 13)` khi mở LMDB state.

Purpose: SC5 (3-service healthy không Go) đang FAIL → chặn luôn SC1 + SC2-browser
(không có backend chạy để render 11 trang React / golden path). Vá gap này mở khoá
toàn bộ phần human UAT còn lại của Phase 8.

Root cause (xác nhận từ `08-HUMAN-UAT.md` + đọc code):
- `api/app/config.py:52` đặt `cocoindex_lmdb_path` default = đường dẫn TƯƠNG ĐỐI
  `Path("Hub_All/.cocoindex/state.lmdb")` — tiền tố `Hub_All/` lạc ngữ cảnh.
- `api/.env` (được `docker-compose.yml` nạp qua `env_file`) đặt
  `COCOINDEX_DB=Hub_All/.cocoindex/state.lmdb`. Trong `api/app/rag/setup.py:64`,
  `os.environ.setdefault("COCOINDEX_DB", ...)` là no-op vì `COCOINDEX_DB` đã có sẵn
  từ `.env` → cocoindex Rust core đọc thẳng `COCOINDEX_DB` lạc này.
- Trong container (`WORKDIR /app`, `USER apiuser` uid 1000), `Hub_All/.cocoindex/...`
  resolve thành `/app/Hub_All/.cocoindex/state.lmdb`. `/app` do `root` sở hữu →
  `apiuser` không tạo được thư mục con → Permission denied.

Cách fix (3 vùng — quyết định của planner, nêu rõ trade-off):
1. config.py + api/.env + .env.example: đổi default/env sang đường dẫn TƯƠNG ĐỐI
   sạch `.cocoindex/state.lmdb` (bỏ tiền tố `Hub_All/`). An toàn cho chạy native
   (uvicorn cwd = `Hub_All/api/` → ghi `Hub_All/api/.cocoindex/`, user sở hữu cwd).
2. docker-compose.yml: container OVERRIDE bằng env tuyệt đối
   `COCOINDEX_DB=/app/.cocoindex/state.lmdb` (`environment:` thắng `env_file:`),
   mount named volume `medinet_cocoindex_state:/app/.cocoindex` để persist.
3. Dockerfile: tạo + chown `/app/.cocoindex` cho `apiuser` TRƯỚC `USER apiuser`.

Trade-off: KHÔNG để default config tuyệt đối `/app/...` (sẽ hỏng chạy native trên
host Windows). Thay vào đó default tương đối an toàn cả 2 môi trường; container ép
đường dẫn tuyệt đối qua env. Đây là pattern "default an toàn + env override" mà
gap_to_close nêu là chấp nhận được.

Output: 6 file sửa thuần api-side / hạ tầng — KHÔNG đụng `frontend/` (D6).

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| host filesystem → container | Bind/named volume mount đưa storage host vào container `python-api` |
| container env → cocoindex Rust core | Biến `COCOINDEX_DB`/`COCOINDEX_LMDB_PATH` điều khiển nơi cocoindex ghi LMDB |
| container `apiuser` (uid 1000) → `/app` (root-owned) | Process runtime không quyền root, chỉ ghi được thư mục đã chown |
