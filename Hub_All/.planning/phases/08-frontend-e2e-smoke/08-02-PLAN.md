Tạo `api/app/routers/ai_chat.py` — router proxy LLM tối giản cho GeminiAssistant (component chat trợ lý, KHÔNG dùng RAG context, khác `/api/ask`).

`APIRouter(tags=["ai"])` KHÔNG prefix (đặt path đầy đủ `/api/ai/chat` như pattern `ask.py`). 1 endpoint:

```python
@router.post("/api/ai/chat")
async def ai_chat(
    request: Request,
    body: AiChatRequest,
    user: User = Depends(get_current_user),  # noqa: B008
) -> JSONResponse:
```

Schema (định nghĩa inline trong file hoặc `api/app/schemas/` — chọn inline cho gọn, Pydantic v2):
- `AiChatMessage`: `role: str`, `content: str`
- `AiChatRequest`: `messages: list[AiChatMessage]`, `system_instruction: str | None = None`
- `AiChatResponse`: `response: str`

Logic:
1. `messages` rỗng → `resp.bad_request(message="messages không được rỗng", code="INVALID_REQUEST")`.
2. Build `llm_messages`: nếu `system_instruction` có → prepend `{"role": "system", "content": system_instruction}`; map từng `AiChatMessage` → dict `{"role", "content"}`.
3. Resolve model: tái dùng `_resolve_llm_model()` từ `app.services.ask_service` (import hàm đó — KHÔNG copy logic; nếu hàm private khó import, gọi `get_settings()` rồi build model string theo cùng pattern `gemini/` prefix). Ghi rõ lựa chọn trong SUMMARY.
4. Gọi `litellm.acompletion(model=..., messages=llm_messages)` non-streaming, bọc try/except: bất kỳ Exception từ litellm → log `ai_chat_llm_failed` (KHÔNG log nội dung message — PII) → `resp.internal_error(message="LLM gọi thất bại", code="LLM_FAILED")` (nếu `resp` không có `internal_error`, dùng helper 500 hiện có — xem `pkg/response.py`, plan 07-04 dùng `resp` cho mã 500; tái dùng đúng helper đó).
5. Trích text trả lời an toàn: `completion.choices[0].message.content` — `getattr` phòng None, fallback chuỗi rỗng.
6. `resp.ok(AiChatResponse(response=answer_text).model_dump())`.

Áp `limiter.limit(SEARCH_LIMIT)` (100/min — tái dùng constant từ `app.middleware`) như `ask.py` để chống abuse.

**Wire:** `api/app/routers/__init__.py` re-export `ai_chat_router` (alias `from app.routers.ai_chat import router as ai_chat_router`). `api/app/main.py` thêm `app.include_router(ai_chat_router)` cạnh `app.include_router(ask_router)` (sau usage_router).
